# EdTech Conversion Funnel Analysis & Automation

### Project Overview
This project focuses on tracking the end-to-end student journey at Cuemath, from the initial **Trial/Demo** request to final **Enrollment (PTM conversion)**. I built these automated pipelines to provide real-time visibility into global funnel performance.

### Key Achievements
* **Automation:** Reduced manual reporting effort by over **70%** using Google Apps Script and automated ETL pipelines.
* **Funnel Optimization:** Identified critical drop-off points between Trial and Demo stages, helping the product team improve conversion rates.
* **Global Insights:** Developed region-wise tracking for India, APAC, EMEA, and ROW stakeholders.

### Tech Stack Used
* **SQL:** Advanced queries involving CTEs, Window Functions, and JSON extraction (Redshift SUPER types).
* **Python:** Used for advanced data cleaning and user segmentation.
* **Automation:** Google Apps Script and Excel/Google Sheets advanced functions.
* **GenAI (Claude):** Leveraged for SQL optimization and structuring complex analytical frameworks.

### How to use this Repository
- `/QL_Queries`: Contains optimized SQL scripts for funnel tracking.
- `ptm_funnel.sql`: Script for tracking Parent-Teacher Meeting outcomes and feedback.
- `trial_mapping.sql`: Logic for mapping trials to successful payments.
